## 역사적 인물 기반 타입 분류

### S급 타입 (통무지정매 중 4개 이상 80+)

#### 1. **천하패자** (All-Rounder Supreme)
**대표**: 조조
- **능력치**: 통99/무72/지91/정97/매96
- **특징**: 모든 분야 능숙, 군사+내정+외교 완벽
- **행동**: 상황에 맞게 최적 선택, 리스크 관리 우수
- **AI 전략**: 기회주의적, 약점 포착 능력

#### 2. **천하명군** (Perfect Ruler)
**대표**: 유비
- **능력치**: 통75/무73/지75/정78/매99
- **특징**: 인망으로 인재 결집, 덕치 중시
- **행동**: 인재 등용 최우선, 민심 관리 집착
- **AI 전략**: 방어적, 동맹 중시, 인재 확보

#### 3. **천하명장** (Perfect General)
**대표**: 관우, 조운, 제갈량
- **능력치 패턴**: 통솔+무력+α 또는 통솔+지력+α
- **특징**: 군사와 전략/무예 겸비
- **행동**: 전쟁 주도, 승리 집착
- **AI 전략**: 공격적, 기회 포착

## 실제 능력치 패턴 분석

### 패턴 1: 완벽형 (조조형)
```
통솔 95+ / 무력 70+ / 지력 90+ / 정치 95+ / 매력 95+
→ 모든 것을 다 잘함
```

### 패턴 2: 명장형 (관우형)
```
통솔 95+ / 무력 97 / 지력 75 / 정치 75 / 매력 95
→ 군사+무예+인망
```

### 패턴 3: 모사형 (제갈량형)
```
통솔 95+ / 무력 30 / 지력 100 / 정치 95+ / 매력 95+
→ 머리+행정+인망
```

### 패턴 4: 맹장형 (여포형)
```
통솔 75 / 무력 100 / 지력 30 / 정치 30 / 매력 30
→ 무력만 최강, 나머지 형편없음
```

### 패턴 5: 군주형 (유비형)
```
통솔 75 / 무력 73 / 지력 75 / 정치 78 / 매력 99
→ 인망으로 승부
```

### 패턴 6: 경세가형 (순욱형)
```
통솔 70 / 무력 30 / 지력 95+ / 정치 99 / 매력 85
→ 내정+계략+명망
```
