# 은하영웅전설 VII 매뉴얼 요약

## 개요

은하영웅전설 VII는 田中芳樹의 SF 소설을 원작으로 한 전략 시뮬레이션 게임입니다. 은하제국과 자유행성동맹 간의 전쟁을 배경으로, 플레이어는 함대를 지휘하여 우주 전투를 수행합니다.

---

## 1. 게임 시작

### 1.1 시나리오 선택
- **Original**: 오리지널 시나리오
- **Novel**: 소설 기반 시나리오

### 1.2 진영 선택
- **은하제국 (Galactic Empire)**: 전제군주제 국가
- **자유행성동맹 (Free Planets Alliance)**: 민주주의 국가
- **페잔 자치령 (Phezzan Dominion)**: 중립 상업 국가

---

## 2. 게임 화면 구성

### 2.1 전략 맵 (Strategic Map)
- 은하계 전체를 조망하는 맵
- 항성계, 요새, 회랑 등 표시
- 함대 이동 및 배치 관리

### 2.2 전술 맵 (Tactical Map)
- 실제 전투가 벌어지는 맵
- 함대 진형 및 전투 명령 수행
- 실시간 또는 턴제 전투

### 2.3 정보 패널
- 함대 상태 (HP, 병력, 사기)
- 제독 정보 (능력치, 스킬)
- 보급 상태

---

## 3. 함대 시스템

### 3.1 함대 구성
- **기함 (Flagship)**: 제독이 탑승하는 지휘함
- **전함 (Battleship)**: 주력 전투함
- **순양함 (Cruiser)**: 기동성 높은 중형함
- **구축함 (Destroyer)**: 소형 고속함
- **항공모함 (Carrier)**: 스파르타니안 탑재
- **공작함 (Engineering Ship)**: 수리 및 지원

### 3.2 함대 편성
- 최대 함대 수: 진영별 상이
- 함대당 최대 함선 수: 약 15,000척
- 제독 1명이 1개 함대 지휘

### 3.3 보급 시스템
- 함대는 보급선 유지 필요
- 보급 부족 시 전투력 감소
- 요새 및 행성에서 보급 가능

---

## 4. 제독 시스템

### 4.1 능력치
| 능력치 | 설명 |
|--------|------|
| **통솔 (Command)** | 함대 지휘 능력, 함대 규모 결정 |
| **전투 (Combat)** | 직접 전투 시 공격력/방어력 |
| **지략 (Intelligence)** | 전술 옵션, 기습 성공률 |
| **정치 (Politics)** | 외교, 내정 능력 |
| **매력 (Charisma)** | 병사 사기, 충성도 영향 |

### 4.2 스킬
- **돌격 (Charge)**: 근접 전투 보너스
- **포격 (Bombardment)**: 원거리 공격 보너스
- **기동 (Maneuver)**: 회피율 증가
- **방어 (Defense)**: 피해 감소
- **지휘 (Leadership)**: 사기 유지

### 4.3 주요 제독
#### 은하제국
- 라인하르트 폰 로엔그람
- 지크프리트 키르히아이스
- 볼프강 미터마이어
- 오스카 폰 로이엔탈
- 파울 폰 오베르슈타인

#### 자유행성동맹
- 양 웬리
- 알렉스 캬젤누
- 더스티 아텐보로
- 올리비에 포플랑
- 월터 폰 쇤코프

---

## 5. 전투 시스템

### 5.1 전투 개시 조건
- 적 함대와 같은 구역 진입
- 요새 공격
- 회랑 돌파

### 5.2 진형 (Formation)
| 진형 | 특징 |
|------|------|
| **어린 (Fish Scale)** | 공격 중시, 돌파력 ↑ |
| **학익 (Crane Wing)** | 포위 공격, 측면 강화 |
| **방원 (Circular)** | 방어 중시, 전방위 대응 |
| **봉시 (Arrowhead)** | 기동력 ↑, 돌격용 |
| **장사 (Long Snake)** | 회피력 ↑, 후퇴 용이 |

### 5.3 전투 명령
- **전진 (Advance)**: 적에게 접근
- **공격 (Attack)**: 사정거리 내 적 공격
- **후퇴 (Retreat)**: 전투 이탈
- **대기 (Hold)**: 현 위치 유지
- **포위 (Encircle)**: 적 측면/후방 공격
- **돌격 (Charge)**: 근접 전투
- **집중포화 (Concentrated Fire)**: 단일 목표 집중

### 5.4 전투 계산
```
데미지 = (공격력 × 함선수 × 진형보정 × 제독보정) - (방어력 × 진형보정)
```

### 5.5 사기 시스템
- 사기 0 = 함대 궤멸/항복
- 승리 시 사기 회복
- 패배/손실 시 사기 감소
- 제독 매력이 사기 유지에 영향

## 7. 특수 시설

### 7.1 이제르론 요새 (Iserlohn Fortress)
- 자유행성동맹 측 요새
- 토르 해머 (Thor Hammer) 장착
- 회랑 방어의 핵심

### 7.2 가이에스부르크 요새 (Geiersburg Fortress)
- 은하제국 측 요새
- 이동 가능 (워프 엔진 장착 시)

### 7.3 페잔 회랑 (Phezzan Corridor)
- 양 진영 연결 통로
- 상업 및 정보 중심지

### 7.4 이제르론 회랑 (Iserlohn Corridor)
- 군사적 요충지
- 주요 전투 발생 지역

## 10. 구현 참고 사항 (개발용)

### 10.1 좌표 시스템
- **전략 맵**: 그리드 기반 (100x50 정도)
- **전술 맵**: 연속 좌표 (10000x10000)
- 3D 고려 시 z축 추가

### 10.2 실시간 전투
- deltaTime 기반 이동
- 50ms tick 권장
- 사정거리 기반 공격

### 10.3 데이터 모델
```typescript
interface Fleet {
  id: string;
  name: string;
  commander: Admiral;
  ships: Ship[];
  strategicPosition: { x: number; y: number };  // 전략 맵
  tacticalPosition: { x: number; y: number };   // 전술 맵
  formation: Formation;
  morale: number;
  supply: number;
}

interface Admiral {
  id: string;
  name: string;
  command: number;    // 통솔
  combat: number;     // 전투
  intelligence: number; // 지략
  politics: number;   // 정치
  charisma: number;   // 매력
  skills: Skill[];
}
```
