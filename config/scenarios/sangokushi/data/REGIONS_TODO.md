# 지역/도시 추가 예정 목록

## 1. 금관가야(김해) 도시 추가 필요 ⚠️ 우선순위: 낮음 (지도 개편 시)
- **이유**: 가야검사(1115) 병종 설명이 "김해 가야의 정예 검사"
- **현재 상황**: 
  * 임시로 계림(신라)에서 징병 가능으로 유지
  * 맵 크기(730x530)가 작아서 한반도 남부에 도시 추가 공간 부족
  * "계림 지역의 가야계 용병" 정도로 해석
- **역사적 배경**: 
  * 삼국지 시대(190-280년) = 금관가야(42-532년) 전성기
  * 대가야는 400년대 이후이므로 시대 맞지 않음
- **추가 시 작업** (지도 리워크 시):
  1. 맵 크기 확장 또는 한반도 지역 재배치
  2. cities.json에 김해, 함안, 고성 등 가야 소국 추가
  3. units.json 가야검사(1115) 제약 변경: reqCity '계림' → '김해'
  4. regions.json에 가야 지역(id:10) 추가 고려

## 2. 선비 도시 추가 필요 ⚠️ 우선순위: 중
- **이유**: 선비족 기마대(1407) 병종이 있으나 선비 도시가 없음
- **현재 상황**: 북방(흉노, 오환)에서 징병 가능하지만 선비 전용 도시 필요
- **제안 위치**: regionId 9 (북방) 또는 regionId 8 (동이) 북부
- **추가 시 작업**:
  1. cities.json에 선비 도시 추가 (적절한 ID, 좌표, 스탯)
  2. regions.json의 북방(id:9) cities 배열에 "선비" 추가

## 3. 동이 지역 도시 추가 (『삼국지』 위서 동이전 기준) ⚠️ 우선순위: 낮음 (지도 개편 시)
- **역사적 근거**: 『삼국지』 위서 동이전(3세기, 280-289년 편찬)에 기록된 세력들
- **추가 가능 도시**:
  1. **부여(夫餘)** - 만주 송화강 유역, 고구려 북쪽
     - 3세기 당시 강력한 세력
     - 위치: 평양/졸본 북쪽
  
  2. **동옥저(東沃沮)** - 함경도 동해안
     - 고구려 동쪽 지역
     - 위치: 이도 북동쪽
  
  3. **읍루(挹婁)** - 만주 동북부
     - 가장 북동쪽 이민족
     - 위치: 맵 최북동단
  
  4. **예(濊)** - 강원도/함경도
     - 동예 지역
     - 위치: 평양-계림 중간 동쪽

- **추가 시 효과**:
  - 역사적 정확성 향상
  - 만주/한반도 북부 지역 다양성 증가
  - 부여/예/옥저 특수 병종 추가 가능

## 4. 기타 확인 사항
- 지역별 밸런스 확인 (일부 지역은 특수 병종이 많고, 일부는 없음)
- 한반도 지역 세분화 고려 (고구려/백제/신라/가야 분리 가능)
