## 🎮 주요 게임 시스템

### 시간 시스템
- **실시간 1분 = 게임 내 24분**
- 실시간 1시간 = 게임 내 1일 (24시간)
- 실시간 1.25일 = 게임 내 1개월 (30일)

### 커맨드 포인트 (CP)
- **PCP** (정략 커맨드 포인트): 정치·운영 능력에 따라 회복
- **MCP** (군사 커맨드 포인트): 지휘·기동·공격·방어 능력에 따라 회복
- 회복: 2게임시간(실시간 5분)마다
- 대용: 부족한 CP는 다른 CP로 2배 소비해 대용 가능

### 계급 시스템
- **12계급**: 소위 → 중위 → 대위 → 소좌 → 중좌 → 대좌 → 준장 → 소장 → 중장 → 대장 → 상급대장(제국) → 원수
- **자동 승진**: 대좌 이하는 게임 내 30일마다 계급 래더 1위 자동 승진
- **인원 제한**: 준장 이상은 인원 제한 (준장 80명, 소장 40명, 중장 20명, 대장 10명, 상급대장 5명, 원수 5명)

### 직무권한 카드
- 모든 커맨드는 직무권한 카드를 통해 실행
- 최소: 개인 카드, 함장 카드
- 최대: 16개까지 겸임 가능

### 승리 조건
1. **결정적 승리**: 인구 90% 지배 + 함정 비율 10배 + 쿠데타 무 + 적 수도 진입(제국)
2. **한정적 승리**: 결정적 승리 조건 중 하나라도 미달
3. **국지적 승리**: 적 수도 점령 외 종료, 인구비 우위
4. **패배**: 상기 조건 모두 열세
