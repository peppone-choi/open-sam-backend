## 주요 발견사항 (별표 외 추출 데이터)

### 1. **복잡한 시간 시스템**
- 실시간 1시간 = 게임 내 24시간 (1일)
- CP 회복: 게임 내 2시간마다 (실시간 5분)

### 2. **이중 커맨드 포인트 시스템**
- PCP (政略): 정치/행정 액션용
- MCP (軍事): 군사 액션용
- 2배 비용으로 대체 사용 가능

### 3. **캐릭터 성장 메커니즘**
- 나이 기반 능력치 변화 + 경험치 기반 성장
- CP 사용 누적으로 경험치 획득 (100 XP = +1 능력치)

### 4. **커맨드 레인지 서클**
- 전술 지휘 반경이 시간에 따라 동적으로 확장
- 각 커맨드 후 0으로 리셋
- 지휘 능력치에 영향받음

### 5. **작전 계획 시스템**
- 3단계: 계획 수립 → 발령 → 실행
- 30일 기간, 완료 시 보너스 포인트

### 6. **정치 영향력 네트워크**
- 사기/혼란/영향력/우호도/지지율/치안율의 복잡한 상호작용
- 게임플레이와 승리에 영향

### 7. **쿠데타 메커니즘**
- 6개 커맨드 반란 시스템
- 설득, 모의, 반란 실행

### 8. **병참 체인**
- 혹성 창고 → 할당 → 부대 창고 → 재편성 → 보충
- 엄격한 매칭 요구사항

### 9. **색적 정보 공유**
- 전술 전투 중 모든 아군이 실시간 색적 정보 공유

### 10. **지상전 박스 시스템**
- 진영당 최대 30유닛 제한
- 혹성/요새 전용 지상전 구역

### 11. **전사는 선택사항**
- 플레이어가 캐릭터 전사 여부 선택 가능
- 기본값: 부상 → 본거지 귀환

### 12. **계급 래더 시스템**
- 각 계급 내 6단계 순위 (공적/작위/훈장/영향력/총능력치 기준)
