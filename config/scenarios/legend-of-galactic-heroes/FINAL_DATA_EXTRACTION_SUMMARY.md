- **PCP (政略)**: 정치/행정 액션용
- **MCP (軍事)**: 군사 액션용
- 2배 비용으로 대체 사용 가능

### 3. 캐릭터 성장 메커니즘
- 나이 기반 능력치 변화
- CP 사용으로 경험치 획득 (100 XP = +1 능력치)
- 60세까지 캐릭터 계승 가능

### 4. 커맨드 레인지 서클
- 전술 지휘 반경 동적 확장 (0-20초)
- 지휘 능력치에 영향
- 각 커맨드 후 0으로 리셋

### 5. 작전 계획 시스템
- 3단계: 계획 수립 → 발령 → 실행
- 30일 기간, 완료 시 보너스 포인트
- 3가지 작전 타입: 점령, 방어, 제압

### 6. 정치 영향력 네트워크
- 사기/혼란/영향력/우호도/지지율/치안율 상호작용
- 게임플레이와 승리 조건에 직접 영향
- 쿠데타 시스템 (6개 커맨드)

### 7. 병참 체인
- 혹성 창고 → 할당 → 부대 창고 → 재편성 → 보충
- 엄격한 매칭 요구사항
- 자동 생산 시스템

### 8. 색적 정보 공유
- 전술 전투 중 아군 전체 실시간 공유
- 거리/정확도 변화
- 은폐 메커니즘

### 9. 지상전 박스 시스템
- 진영당 최대 30유닛 제한
- 별도 전투 처리
- 경험치/사기 시스템

### 10. 전사는 선택사항
- 플레이어가 전사 허용/거부 선택
- 기본값: 부상 → 본거지 귀환
- 회복 기간 존재

### 11. 계급 래더 시스템
- 각 계급 내 6단계 순위
- 공적/작위/훈장/영향력/총능력치 기준
- 자동 승진 판정

### 12. 함선 생산 체계
- 제국: SS75(전함), PK86(고속전함), SK80(순양함), Z82(구축함)
- 동맹: 787년형(전함), 795년형(순양함), 794년형(타격순양함), 796년형(구축함)
- 각 함급 8개 변형 (I~VIII형)
