## 데이터 구조

모든 시나리오는 `sangokushi` 폴더의 기본 데이터를 공유합니다:
- **도시**: `../sangokushi/data/cities.json`
- **장수**: `../sangokushi/data/generals.json` (시나리오별 초기 배치만 다름)
- **상수**: `../sangokushi/data/constants.json`
- **병종**: `../sangokushi/data/units.json`
- **맵**: `../sangokushi/data/map.json`

시나리오별 고유 데이터 (`data.scenario`):
- **국가 초기 배치**: `nations` - 초기 국가, 도시 소유, 국고, 정책 등
- **장수 초기 배치**: `generals` - 초기 소속 국가, 도시, 능력치 등
- **외교 관계**: `diplomacy` - 국가 간 불가침, 동맹 등
- **이벤트**: `events` - 시나리오별 이벤트 트리거
- **역사 기록**: `history` - 초기 역사 로그

## 사용 방법

```typescript
// 세션 생성 예시
await SessionService.createSessionFromScenario(
  'sangokushi-heroes',      // 시나리오 ID
  'heroes_room1',           // 세션 ID
  '군웅할거 방 1',           // 세션 이름
  true                      // 자동 초기화
);
```

## 외교 관계 형식

외교 데이터는 `data.scenario.diplomacy` 배열에 저장되며, 각 항목은:
```json
{
  "id": 1,
  "from": 1,      // 국가 ID (발신)
  "to": 2,        // 국가 ID (수신)
  "type": 1,      // 1=불가침, 7=동맹 등
  "term": 36      // 유지 기간
}
```

## 참고사항

- **외교 데이터**: 대부분의 시나리오에 포함되어 있습니다 (0개인 시나리오는 초기 외교 관계가 없는 경우)
- **장수 수**: 대부분 491개 또는 680개 (정사 버전은 더 많은 장수 포함)
- **국가 수**: 시기별로 2~24개로 다양함
